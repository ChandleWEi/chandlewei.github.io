height__100%_问题_(FF)

Feb 19th, 2008 | Comments

Firefox浏览器中使用“height: 100%”会固定其高度，而不是自动延伸成100%的高度，这个在IE浏览器中非常非常正常的一个CSS定义，到Firefox浏览器中就会有问题,

 但是如果使用以下头定义

<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd”>

则会使不符合w3c的定义失效，所以今天碰的问题是

div main 设置了100％，但是被firefox 固定了高度，所以第三层的 footer 的margin－top 出现问题。
