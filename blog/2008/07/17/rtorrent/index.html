Rtorrent

Jul 17th, 2008

rtorrent 使用
Category: Linux/Unix, Author: soai, Popularity: 28% [?]
※ 以下含有 ^ 字符的代表Ctrl鍵

QUOTE:
常用熱鍵

  1. 方向鍵 : ↑,↓,←,→
  2. Torrent 開始下載 : Ctrl + S (Screen下用法為: Ctrl+ a + s)
  3. Torrent 暫停下載 : Ctrl + D
  4. Torrent 停止下載 : Ctrl + K
  5. 清除Torrent下載列表 : [法1] 在停止狀態使用Ctrl + D, [法2] 在下載完成連續執行 Ctrl+D 二次
  6. 限定最大上傳速度 : [增加]a / s / d [減少] z / x /c
  7. check file hash : Ctrl + R
  8. 離開rtorrent程式 : Ctrl+ q (Screen下用法為 : Ctrl+ a + q)

QUOTE:
全域熱鍵 (Global Keys)
目地 : 用來控制系統操作,全域的設定,最大速度,程式的關閉…等等。命令
說明

^q 關閉程式

QUOTE:
注意 : 在screen內用法,先按Ctrl+a 然後再按 q

up / down 選擇項目上/下
← 返回之前一頁
a | s | d 增加全域最大上傳的速度 by 1/5/50 KB. 增加下圖(1)的值
A | S | D 增加全域最大下載的速度 by 1/5/50 KB. 增加下圖(2)的值
z | x | c 減少全域最大上傳的速度 by 1/5/50 KB. 減少下圖(1)的值
Z | X | C 減全域最大下載的速度 by 1/5/50 KB. 減少下圖(1)的值

可用命令 :命令
說明

→ 觀看此種子下載狀態.

View torrent file list. Use the space-bar to change the file priority and * to change the priority of all files.
1 – 7 切換檢視模式
Change view.

1 : 顯示全部下載 Show all downloads
2 : 以名稱排序 Show all downloads, ordered by name
3 : 目前啟動的 Show started downloads
4: 目前停止中的 Show stopped downloads
5 : 已完成的 Show complete downloads
6 : 未完成的 Show incomplete downloads
7 : hash? Show hashing downloads
^S 開始下載
Start download.

QUOTE:

注意 : 在screen內用法,先按Ctrl+a 然後再按 s

^D 停止一個正在啟動的下載,或移除一個已停止的下載
使用停止功能時會出現如圖1的(2)

Stop an active download, or remove a stopped download.
^K 關閉一個Torrent檔及其檔案

Close a torrent and its files.
^R 開始執行此Torrent檔案的hash檢查

Initiate hash check of torrent.
^O 改變下載的目地目錄。 使用前此torrent必須是關閉的 (^K)

Change the destination directory of the download. The torrent must be closed.
+ | – 改變下載的優先權 (一共有auto,low,off,high 四種模式)

Change the priority of the download.
backspace Add torrent using an URL or file path. Use tab to view directory content and do auto-complete.
l 觀看記錄檔,用空白鍵離開 (在第一頁才有效)

View log. Exit by pressing the space-bar.
^P 呼叫命令或是改變設定

Call commands or change settings.
U 刪除此torrent關聯的torrent檔案案,,並清除關連

Delete the file the torrent is tied to, and clear the association.

I Toggle whether torrent ignores ratio settings.
