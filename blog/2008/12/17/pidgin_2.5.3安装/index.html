pidgin_2.5.3安装

受够了 debian 的 pidgin 2.4.3的问题去官方网站一看我考都，2。5。3了赶快弄下来编译

  CFLAGS=-O3 ./configure –disable-nm

make 

make check

make install

sudo ldconfig

万事ok, 2.5.2开始了
