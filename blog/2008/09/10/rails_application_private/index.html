It’s Ruby. Syntax has different meaning than in Java. In Rails you use
private in controllers to mean I don’t want this method to be called from
outside.

http://en.wikibooks.org/wiki/Ruby_Programming/Classes#Private

irb(main):001:0> class T
irb(main):002:1> private
irb(main):003:1> def x
irb(main):004:2> puts “x”
irb(main):005:2> end
irb(main):006:1> end
=> nil
irb(main):007:0> y = T.new
=> # irb(main):008:0> y.x NoMethodError: private method `x’ called for #         from (irb):8         from :0 irb(main):009:0> class W < T irb(main):010:1> def q irb(main):011:2> x irb(main):012:2> end irb(main):013:1> end => nil irb(main):015:0> z = W.new => # irb(main):016:0> z.q x => nil irb(main):017:0>
