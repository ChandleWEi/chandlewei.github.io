Ruby实例变量

Jun 5th, 2008 | Comments

http://sporkmonger.com/2007/2/19/instance-variables-class-variables-and-inheritance-in-ruby

 

class ExampleClass
@variable = “foo”
@@variable = “bar”

def initialize
@variable = “baz”
end

def self.test
puts @variable
end

def test
self.class.test
puts @@variable
puts @variable
end
end

class ExampleSubclass < ExampleClass
@variable = “1”
@@variable = “2”

def initialize
@variable = “3”
end
end

first_example = ExampleClass.new
first_example.test

puts “—”

second_example = ExampleSubclass.new
second_example.test
