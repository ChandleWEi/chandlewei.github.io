计划与变化

我们曾经有过很多次计划，旅行、看书、学英文…

如你所知，计划一开始就面临着变化…天气、请客吃饭、好不容易的节假日应该放松、今天工作比较多看书暂缓、今天受了刺激，必须先学英文、最近没钱旅行延迟….

我不是抱怨额…我是个没心没肺的人，想到什么说什么而已…我也向来不是个理智与自控的人，对于我来说，随心所欲的生活，那真是像进入天堂…我好多的理智观念，来自于你的熏陶，因为我们天天黏在一起已经七百多天将近800天了，而且你一定不知道你对于我的想法影响之深厚啊…

所以，我想说什么呢？两个人在一起，他们的计划面临着两个变量，要执行，是两个人的自控与互相监督。

或者，换一个说法吧，也许，应该是互为动力。当你懈怠的时候，想想我，能不能又力量暴增，当我无聊发呆的时候，想想你，会不会小宇宙爆发？这是多理想的状况啊…

我们黏在一起的时间实在是太长了，有时候我也反省，我们一起吃饭，一起逛街，一起买东西，一起拍照，这些享乐式的快乐并不是生活的本质吧，因为人终究是应该上进和付出的，那么我们一起努力的事情，好像没有多少成功的案例啊..我们相处的成就感何来呢？难道我们未来会不如庸庸碌碌的家庭生活吗？这不是我们彼此希望的结局吧..

我们那些惰性的因子，本身并不能为我们带来快乐。真正的快乐是成就感、存在感吧。比如通过努力英文增强、iphone开发学成，比如通过努力存下钱来一起去旅游，比如，通过努力为你织了件围脖（这个不用想了，目前我没这打算）…

明白了这些，我觉得任何牺牲都是值得的吧，因为一起付出的过程其实比结果还要幸福，我们过得并不苦啊…呵呵。

所以，我们确实应该多为自己打鸡血。就像我自己多写写类似的小文？哈哈。写完我觉得很有动力。真的。希望你看了也一样。

解决spotlight无法索引的问题[z]

 

从昨天开始发现使用系统自带的spotlight查找文件时出现问题。开始是找的特别慢，完全没有往日的闪电侠风采，今天干脆找不到了，风火轮不停的转，msd进程占cpu的30%。猜测可能是索引出问题了，于是采用如下命令重建索引：
sudo mdutil -i off /
该命令用来关闭索引
sudo mdutil -E /
该命令用来删除索引
sudo mdutil -i on /
该命令用来重建索引
然后用快捷键呼出spotlight菜单，随便输入一个词，就能看到提示，正在进行索引，并且告诉你重建索引的时间。

完成之后，spotlight又可以运转如飞了。
