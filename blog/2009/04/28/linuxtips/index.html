echo “Hello world” base64

Encryption with base64

Also works with files:

$ cat file

Hello world

$ base64 file

SGVsbG8gd29ybGQK

To decrypt use the -d option:

$ echo SGVsbG8gd29ybGQK | base64 -d

Hello world
